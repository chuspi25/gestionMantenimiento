# Producto

Aplicación web full-stack desarrollada con tecnologías modernas y arquitectura limpia.

## Características principales
- Frontend: HTML5, CSS3, TypeScript (compilado a Vanilla JS)
- Backend: Node.js con Hono framework
- Autenticación: JWT + bcrypt para hash de contraseñas
- Base de datos: Configuración flexible mediante variables de entorno
- Testing: Suite de pruebas completa para garantizar funcionalidad

## Objetivos
- Código limpio y bien estructurado
- Separación clara entre frontend y backend
- Seguridad robusta con autenticación JWT
- Facilidad de despliegue y mantenimiento
